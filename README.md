# Put It Out There

Polyglot release orchestrator for single-maintainer, LLM-authored projects
that publish to crates.io, PyPI, and npm from one monorepo.

> **Status.** v0 implementation in progress. See the [v0 epic](https://github.com/thekevinscott/put-it-out-there/issues/2) for the full work breakdown.

User-facing README (installation, quickstart, worked example) lands in
[#32](https://github.com/thekevinscott/put-it-out-there/issues/32) once
the CLI is functional.
